# Tiny iLvL
An extremely light-weight addon for World of Warcraft that simply adds the item level at the corner of the inspect frame.

So far it has been tested and works on both Patch 12.0.5 of Midnight and Patch 5.5.3 of MoP Classic

## Installing
To install this addon simply go to [the page for the latest release](https://github.com/fabriond/Tiny_iLvL/releases/latest) and download the Tiny_iLvL.zip and extract it into your WoW addons folder.
